# &lt;E2E/&gt; Mentoria

- Página para mentoria de carreira internacional para engenheiros brasileiros
- De engenheiro para engenheiro
- Preparação técnica real com quem acabou de fazer a travessia do Brasil para a Europa

## Tecnologias

- HTML + CSS puro
- Tema blueprint (Inter, grid 40×40px)
- Sem frameworks ou dependências

## Estrutura

```
index.html       # Página principal
404.html         # Página de erro
css/
└── style.css    # Estilos
.nojekyll        # Desativa Jekyll no GitHub Pages
README.md
```

## Deploy

- Publicado via GitHub Pages a partir da raiz do branch principal.
